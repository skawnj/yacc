# SQLite 스키마 개요(2016-06-28 14:18 기준)
* RESTR_BASE 식당기본
	- RID TEXT PRIMARY KEY 식당ID
	- NAME TEXT 이름
	- ADDR TEXT 주소
	- PHONE TEXT 전화번호
	- COORD_X REAL X좌표
	- COORD_Y REAL Y좌표
	- DIST_HQ REAL 본점거리
	- DIST_IT REAL 별관거리
	- UPT_TIME TEXT 갱신시각
	- AVG_RATING REAL 평균평점
	- HIT_SCORE INTEGER 히트점수
	- DCODE_URL TEXT 다이닝코드URL
	- THUMBNAIL_URL TEXT 썸네일이미지URL
* USER_BASE 사용자기본
	- SID TEXT PRIMARY KEY 세션ID
	- NICKNAME TEXT 별명
	- REG_TIME TEXT 등록시각
	- LAST_CONN_TIME TEXT 최종접속시각
* USER_REVIEW 사용자리뷰
	- SID TEXT 세션ID
	- RID TEXT 식당ID
	- REVIEW_TIME TEXT 리뷰시각
	- RATING REAL 평점
	- REVIEW_TXT TEXT 리뷰텍스트
	- PRIMARY KEY(SID, RID, REVIEW_TIME)
	- FOREIGN KEY(SID) REFERENCES USER_BASE(SID)
	- FOREIGN KEY(RID) REFERENCES RESTR_BASE(RID))
* USER_VIEW_HIST 사용자조회내역
	- SID TEXT 세션ID
	- RID TEXT 식당ID
	- VIEW_TIME TEXT 조회시각
	- STAY_SECONDS INTEGER 체류초
	- PRIMARY KEY(SID, RID, VIEW_TIME)
	- FOREIGN KEY(SID) REFERENCES USER_BASE(SID)
	- FOREIGN KEY(RID) REFERENCES RESTR_BASE(RID))
